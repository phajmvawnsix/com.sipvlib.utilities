# com.sipvlib.utilities

Part of [SiPVLib]. General-purpose singleton base classes (`Singleton<T>`, `MonoSingleton<T>`), extension methods for classes, `long` (Unix time), and `GameObject`/`Component`, plus misc helpers like exponential retry backoff and the `SubAsset` attribute for ScriptableObject sub-assets.

## Install

Add to your project's `Packages/manifest.json`:

```json
"com.sipvlib.utilities": "https://github.com/phajmvawnsix/com.sipvlib.utilities.git"
```

No dependencies.
